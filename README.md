Digital Literacy Project
Project Overview

This project was created as part of the Digital Literacy course to improve digital awareness, professional communication skills, and online safety knowledge. The main objective of this project is to help students understand how to use digital tools effectively, communicate professionally through email, create useful digital content, and stay safe while using the internet.

The project includes a presentation, professional profiles, platform-based tasks, email writing, and a cybercrime awareness case study. It focuses on both technical skills and responsible digital behaviour.

Project Tasks
Task 1 – Presentation

A digital literacy presentation slide was created using Canva. The slide explains what digital literacy is, useful digital tools for students, safe internet practices, and professional online presence. This task helped improve creativity and presentation skills.

Task 2 – Portfolio Creation

Professional profiles were created on the following platforms:

LinkedIn
Kaggle
ResearchGate

The purpose of this task was to build a strong professional online presence and prepare for future internships and learning opportunities.

Task 3 – Platform Practice

This task included two parts:

Part A: Created an account on HackerRank to practise coding and improve problem-solving skills.
Part B: Created a Google Form titled “Digital Literacy Awareness Quiz” with five questions.

Task 4 – Email Etiquette & Social Media Responsibility

Part A: Two professional emails were written:

Email to a professor requesting an assignment extension
Email to an internship coordinator expressing interest in a summer internship

Part B: A checklist was created showing the Do’s and Don’ts of responsible social media use for college students.

Task 5 – Cybercrime Awareness

Part A: A case study was written on UPI / Online Payment Fraud, explaining how the crime happens, who is targeted, and the consequences.

Part B: A “Stay Safe Online” checklist was created with important online safety tips, including UPI and financial safety.
