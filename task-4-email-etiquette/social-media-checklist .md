#            Social media checklist

 Do's and  Don'ts for responsible social media use as a college student. 

|                             Do's |                            Don'ts  |
| :---- | :---- |
| Think before posting | Don’t share personal info |
| Use strong passwords | Don’t post harmful content |
| Respect others online | Don’t spread fake news |
| Share useful content | Don’t waste time scrolling |
| Keep accounts secure | Don’t engage in cyberbullying |
